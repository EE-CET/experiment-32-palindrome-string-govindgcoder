# Experiment 32: Palindrome String

## Problem Statement

Check whether a given string is a palindrome.
A palindrome is a string that reads the same forward and backward.

**Note:** The check should be **case-insensitive** (e.g., 'M' and 'm' are considered matching).

## Input Format

* A single string.

## Output Format

* Print `1` if the string is a palindrome.
* Print `0` otherwise.

### Example 1

**Input:**

```text
Malayalam
```

**Output:**

```text
1
```

### Explanation
"Malayalam" reads the same backwards (ignoring case), so the output is 1.
